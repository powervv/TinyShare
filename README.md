# TinyShare ✨

一款极简、轻量级且极具现代设计感的网页端文本与文件共享应用。无需安装任何客户端，只需在浏览器中打开特定网址，即可实现局域网或公网下的多端实时数据共享。

## 🌟 核心功能
* **随机房间号自动分配**：直接访问首页将自动进入一个随机字符串生成的隐私房间。
* **暗黑“毛玻璃” UI (Glassmorphism)**: 原生 HTML/CSS 实现的深邃高级界面，带来出色的视觉体验。
* **动态双向同步**：仿聊天室布局的消息面板与文件共享面板，数据收发自动无刷新拉取。
* **急速拖拽上传**：由底层 API 及 Multer 驱动的文件流上传功能。
* **轻量级本地化持久存储**：数据归档至本地 `data/` 目录，随开随用，免去数据库配置烦恼。

## 🚀 快速启动

### 1. 环境准备
请确保您的设备已安装了 [Node.js](https://nodejs.org/)。

### 2. 安装项目依赖
在终端中进入项目根目录并执行以下命令安装依赖：
```bash
npm install
```

### 3. 运行服务
启动该应用：
```bash
npm start
```
或者直接使用 node：
```bash
node server.js
```

### 4. 访问系统
服务默认运行在 `3000` 端口。请在浏览器中访问：[http://localhost:3000](http://localhost:3000)
